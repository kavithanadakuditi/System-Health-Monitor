\# System Health Monitor



\## 📌 Description

This project is a Python-based tool developed to monitor system performance. It helps users check important system metrics such as CPU usage, memory usage, and disk usage.



\## 🚀 Features

\- Displays CPU usage

\- Shows memory usage (total, used, percentage)

\- Displays disk usage details

\- Simple menu-based interface



\## 🛠️ Technologies Used

\- Python

\- psutil library



\## ▶️ How to Run

1\. Open Command Prompt in the project folder

2\. Run the command:

&#x20;  py health\_monitor.py



\## 🎯 Use Case

This tool helps monitor system performance and identify resource usage issues, similar to tasks performed by IT support professionals.
